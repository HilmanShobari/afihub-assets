# AFIHUB Assets

Repository ini digunakan untuk menyimpan seluruh asset visual, mockup desain, screenshot, dan referensi UI/UX untuk portal brand **AFIHUB**.

## Struktur Folder yang Direkomendasikan:
- `/auth/` - Asset halaman Login dan Register.
- `/onboarding/` - Asset langkah-langkah Onboarding Brand (Identity, Sales Channels, Social Presence, AI Persona).
- `/products/` - Asset modul CRUD Produk (Daftar Produk, Tambah Produk, Detail Produk).
- `/branding/` - Logo, ikon, palet warna, dan aset branding utama AFIHUB.

## Panduan Kontribusi Asset:
1. Pastikan penamaan file menggunakan format snake_case (contoh: `login_page_mockup.png`).
2. Gunakan format file yang dioptimalkan (PNG/WebP untuk screenshot, SVG untuk ikon/logo).
